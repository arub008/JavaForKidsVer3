Что такое Java?

Это язык программирования.

Когда он появился? Первая версия вышла в 1995-м году. У первой версии было кодовое имя oak.

Для чего разрабатывался? Изначально использовался для программирования бытовых электронных устройств, но позже стал применяться для написания различных приложений и серверного программного обеспечения.

Почему Java?

Язык получил такое название в честь марки кофе Java, которая в свою очередь получила свое название от одноимённого острова Ява, поэтому на официальной эмблеме языка изображена чашка с парящим кофе.

Какими особенностями обладает язык Java

Во время компиляции программы транслируются в байт код. Он выполняется виртуальной машине Java - специальной программой, которая обрабатывает этот байтовый код и передает инструкции оборудованию как интерпретатор. В этом заключается большое преимущество языка Java, т.к. получается, что байт код программы полностью независим от операционной системы и оборудования. Это позволяет выполнять Java-приложения на любом устройстве, для которого существует виртуальная машина Java.

Язык Java очень активно применяется для создания мобильных приложений под операционную систему андроид. При этом программа компилируется в нестандартный байт код для использования их виртуальной машиной Dalek (фанаты Doctor Who есть?).
